# my_to_do_list

Este repositório foi criado para gerenciar uma lista de tarefas simples usando Python. Ele permite adicionar, concluir, listar e limpar tarefas concluídas.

O objetivo é praticar conceitos básicos de programação em Python, bem como ativar o uso do GitHub Actions para integração contínua nas versões do Python 3.9, 3.11 e 3.12.
